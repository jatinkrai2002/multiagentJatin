 GitHub Actions Pipeline Overview
The pipeline will:
    Trigger on pushes or pull requests to main.
    Build & test the Python orchestrator and agents.
    Build Docker images for orchestrator and agents.
    Push images to GitHub Container Registry (GHCR).
    Deploy using docker-compose or Kubernetes (simulated enterprise-scale).

.github/workflows/ci-cd.yml


Enterprise-Scale Simulation
    Scaling agents: In docker-compose.yml, you can scale agents with docker-compose up --scale analyzer=3.
    Monitoring: Add Prometheus/Grafana containers to monitor agent health.
    Kubernetes option: Replace docker-compose with Helm charts for orchestrator + agents.
